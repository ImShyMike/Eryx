TestLang WebUI

TestLang is a simple programming language that is designed to be easy to learn and use. It is a dynamically typed language that supports basic data types, control structures, and functions. The language is interpreted and runs on a stack-based virtual machine.

Basic Data Types

  • Integer
  • Boolean
  • Dictionary

Functions

  • Function Definition
  • Function Call

Operators

  • Arithmetic Operators
  • Assignment Operators

Comments

TestLang supports single-line comments with the '#' character.

Examples

Data types 

const timenow = time();

func add (x, y) {
    let result = x + y;

    func sub (a, b) {
        a - b
    }

    let result2 = sub(y, x);

    let obj2 = {
        x,
        y
    };

    let obj = {
        subResult: result2,
        timenow,
        result,
        object: obj2
    };

    obj
}

print(add(5, 7))
